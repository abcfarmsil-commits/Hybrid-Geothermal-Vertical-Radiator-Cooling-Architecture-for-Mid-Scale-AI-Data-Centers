## Phase 3: Frame Construction (Week 2)

### Safety First
- Wear safety glasses and work gloves
- Use proper lifting techniques (rack will be heavy)
- Ensure adequate ventilation if cutting metal
- Use clamps to secure pieces during assembly
- Have first aid kit accessible

### Step-by-Step Assembly

#### Step 1: Base Frame Assembly
**Time Estimate:** 2-3 hours

1. **Layout base components:**
   - 2 pieces @ 25.5" (sides)
   - 2 pieces @ 18.8" (ends)
   - 4 caster plates

2. **Attach casters to base pieces:**
   - Position casters at each corner
   - Use appropriate bolts and lock washers
   - Tighten securely but don't overtighten (caster damage)
   - Ensure locking casters are on same side

3. **Assemble base rectangle:**
   - Connect sides to ends using corner brackets
   - Square corners (measure diagonals - should be equal)
   - Clamp and secure with appropriate fasteners
   - Double-check squareness before final tightening

**Quality Check:**
- Base sits level on floor (use bubble level)
- All casters roll smoothly
- Locking casters engage properly
- Frame is square (diagonal measurements equal)

#### Step 2: Upright Installation
**Time Estimate:** 1-2 hours

1. **Mark upright positions on base:**
   - Measure and mark 4 corner positions
   - Ensure alignment with caster positions
   - Mark center points for drilling

2. **Attach uprights to base:**
   - Use corner brackets or L-brackets for reinforcement
   - Drill pilot holes to prevent metal cracking
   - Secure with bolts and lock washers
   - Use torque wrench for consistent tightness

3. **Verify plumb (vertical alignment):**
   - Use bubble level on each upright
   - Adjust as needed before final tightening
   - All uprights should be perfectly vertical

**Quality Check:**
- Uprights are plumb (vertical)
- All uprights are same height
- Connections are secure and tight
- No wobble or instability

#### Step 3: Top Frame Installation
**Time Estimate:** 1-2 hours

1. **Assemble top rectangle:**
   - Same dimensions as base (25.5" x 18.8")
   - Use same assembly process
   - Ensure squareness

2. **Install top frame on uprights:**
   - Carefully position on top of uprights
   - Use assistant if possible (top is awkward alone)
   - Secure with brackets at each corner
   - Double-check alignment with base

**Quality Check:**
- Top frame is level
- Top frame aligned with base
- Entire structure is square and plumb
- Structure is stable with no wobble

#### Step 4: Cross Bracing Installation
**Time Estimate:** 1 hour

1. **Install mid cross braces:**
   - 2 braces @ 18.5" (front and rear)
   - Position at vertical midpoint
   - Use appropriate brackets
   - Ensure braces don't interfere with tray slides

2. **Install rear tray stops (optional):**
   - 16 stops @ 15.5" length
   - Position at each level
   - Provides backstop for trays
   - Use aluminum flat bar 1" x 1/8"

**Quality Check:**
- Braces add rigidity without interfering
- Structure feels solid and stable
- No flexing when pressure applied
- All connections secure

## Phase 4: Tray Slide Installation (Week 2-3)

### Objective
Install 32 tray slide angles (2 per level) for smooth tray operation

### Installation Steps

#### Step 1: Mark Level Positions
**Time Estimate:** 2 hours

1. **Calculate level spacing:**
   - Total growing height: 68" - 6" (base) - 6" (top) = 56"
   - 16 levels = 56" / 16 = 3.5" spacing per level
   - First level: 6" from base
   - Subsequent levels: Every 3.5"

2. **Mark positions on both uprights:**
   - Use measuring tape and level
   - Mark consistent heights on left and right
   - Double-check measurements
   - Use pencil initially (adjustable)

**Quality Check:**
- All level positions marked
- Left and right marks align perfectly
- Spacing is consistent
- First and last levels positioned correctly

#### Step 2: Install Tray Slides
**Time Estimate:** 4-6 hours

1. **Install bottom pair first:**
   - 2 angles @ 23" length (1" x 1" x 1/16")
   - Position at bottom level marks
   - Use brackets or through-bolts
   - Ensure level (left-right and front-back)

2. **Work upward systematically:**
   - Install one level at a time
   - Use installed levels as reference
   - Maintain consistent spacing
   - Check level after each installation

3. **Test slide operation:**
   - After installing 3-4 levels, test with actual tray
   - Tray should slide smoothly
   - No binding or sticking
   - Adequate clearance for drainage

**Quality Check:**
- All 32 slides installed
- Each slide level and secure
- Trays slide smoothly on all levels
- No interference between levels

#### Step 3: Install Tray Stops
**Time Estimate:** 1-2 hours

1. **Install rear stops on each level:**
   - 16 stops @ 15.5" length
   - Position at back of each slide
   - Prevents trays from sliding off back
   - Use small brackets or rivets

**Quality Check:**
- All stops installed
- Trays stop at consistent position
- No damage to tray edges
- Easy tray removal and insertion

## Phase 5: LED Lighting Installation (Week 3)

### Objective
Install 16 LED bars with proper wiring and mounting

### Pre-Installation Checks
- [ ] Verify LED bar specifications match power supply
- [ ] Test each LED bar before installation
- [ ] Confirm 24V DC power supply capacity
- [ ] Plan wiring route for clean installation

### Installation Steps

#### Step 1: Mount LED Bars
**Time Estimate:** 3-4 hours

1. **Position LED bars:**
   - Mount to underside of slide above each level
   - Use 15° tilt brackets for optimal light angle
   - Center bars over growing area
   - Maintain consistent spacing

2. **Secure mounting:**
   - Use appropriate brackets and fasteners
   - Ensure bars are parallel to tray slides
   - Double-check alignment with growing trays
   - Tighten securely but don't damage LED housing

3. **Test positioning:**
   - Temporarily power one LED bar
   - Verify light coverage over tray area
   - Check for shadows or hotspots
   - Adjust angle if needed

**Quality Check:**
- All 16 LED bars installed
- Bars are parallel and level
- Light coverage is uniform
- No mechanical interference with trays

#### Step 2: Wiring Installation
**Time Estimate:** 3-4 hours

1. **Plan wiring route:**
   - Route wiring along back of rack
   - Use cable management clips or ties
   - Keep wiring away from water and irrigation
   - Plan for future maintenance access

2. **Install wiring harness:**
   - Use 24V DC trunk with quick-disconnects
   - Connect each LED bar to trunk
   - Use proper gauge wire for current capacity
   - Include inline fuses for safety

3. **Connect to power supply:**
   - Mount power supply at top or bottom of rack
   - Ensure proper polarity
   - Include main power switch
   - Test all connections before final securing

**Quality Check:**
- All wiring secure and organized
- No loose connections
- Proper polarity observed
- Wiring protected from water damage
- Easy access for maintenance

#### Step 3: Drip Guard Installation
**Time Estimate:** 1-2 hours

1. **Install drip guards:**
   - 16 clear polycarbonate guards (~3" x 22")
   - Position above each LED bar
   - Protect LEDs from water and debris
   - Use small brackets or adhesive

**Quality Check:**
- All drip guards installed
- No obstruction of light
- Protection from above water
- Easy to clean and maintain

## Phase 6: Irrigation System Installation (Week 3-4)

### Objective
Install mist irrigation system for automated watering

### System Design
- **Method:** Mist irrigation with anti-drip nozzles
- **Pressure:** 30-40 psi (regulated)
- **Flow:** 7-10 L/hr per nozzle
- **Coverage:** 3 misters per level (48 total)

### Installation Steps

#### Step 1: Install Vertical Header
**Time Estimate:** 1-2 hours

1. **Mount 1/2" PEX or PVC header:**
   - Run vertical along back of rack
   - Position near LED wiring (organized together)
   - Secure with clips or brackets every 12-18"
   - Ensure accessibility for maintenance

2. **Install solenoid valve:**
   - Mount at base of vertical header
   - Use 24VDC, 3/4" NPT valve
   - Connect to timer/controller
   - Test operation before final securing

**Quality Check:**
- Header securely mounted
- No kinks or sharp bends
- Solenoid valve accessible
- Leak-free connections

#### Step 2: Install Pressure Regulation
**Time Estimate:** 1 hour

1. **Install pressure regulator:**
   - Mount inline before solenoid valve
   - Set to 30-40 psi
   - Include pressure gauge for monitoring
   - Test pressure settings

2. **Install filter:**
   - 100-150 mesh inline filter
   - Position before pressure regulator
   - Protect mister nozzles from debris
   - Make accessible for regular cleaning

**Quality Check:**
- Pressure regulated correctly
- Filter installed and accessible
- No leaks in connections
- Proper pressure maintained

#### Step 3: Install Quick-Connect
**Time Estimate:** 30 minutes

1. **Install dry-break quick-connect:**
   - At main water connection point
   - Use 3/4" or 1/2" fitting
   - Allows easy rack disconnection
   - Test connection and disconnection

**Quality Check:**
- Quick-connect operates smoothly
- No leaks when connected/disconnected
- Easy to operate
- Secure connection when locked

#### Step 4: Install Level Mist Lines
**Time Estimate:** 4-6 hours

1. **Install 1/4" poly lines at each level:**
   - 16 lines, one per level
   - Connect to vertical header with tees
   - Run along back of tray area
   - Secure with clips or ties

2. **Install mister nozzles:**
   - 3 nozzles per level (48 total)
   - Space evenly along level
   - Position to spray toward growing area
   - Test spray pattern

**Quality Check:**
- All 48 misters installed
- Even coverage on each level
- No overlapping or dry spots
- Misters positioned correctly
- Easy to replace if needed

#### Step 5: System Testing
**Time Estimate:** 1-2 hours

1. **Pressure test entire system:**
   - Close all nozzle outputs temporarily
   - Pressurize system to 40 psi
   - Check for leaks at all connections
   - Fix any leaks before proceeding

2. **Test mist operation:**
   - Open nozzles and test operation
   - Verify uniform coverage on each level
   - Check for proper mist pattern
   - Adjust nozzle positions as needed

**Quality Check:**
- No leaks anywhere in system
- Uniform mist coverage
- Proper spray pattern
- System operates reliably
- Timer/controller functions correctly

   - Test locking mechanism

- No noticeable flexing or stress
- Casters operate smoothly under load
- Locking mechanism works reliably

#### Test 2: Lighting System

1. **Individual LED test:**

3. **Cycle test:**

   - Verify consistent operation
   - Check for any system fatigue or issues

**Pass Criteria:**
- No leaks in irrigation system
- Uniform water coverage on all levels
- Proper spray pattern and coverage
- System operates reliably over multiple cycles
- Timer/controller functions correctly

#### Test 4: Integrated Systems Test
**Time Estimate:** 2-3 hours

1. **Simulate growing cycle:**
   - Run complete lighting and irrigation cycles

   - Monitor all system parameters

## Troubleshooting Guide

### Common Issues and Solutions

#### Structural Issues
**Problem:** Rack wobbles or flexes
- **Solution:** Add additional cross bracing
- **Prevention:** Ensure all connections are tight and square

**Problem:** Casters don't roll smoothly
- **Solution:** Check for debris in wheels, lubricate if needed
- **Prevention:** Regular cleaning and maintenance

#### Lighting Issues
**Problem:** LEDs not turning on
- **Solution:** Check power supply, verify wiring connections
- **Prevention:** Test each LED before installation

**Problem:** Uneven light coverage
- **Solution:** Adjust LED bar positioning or angle
- **Prevention:** Test coverage before final installation

#### Irrigation Issues
**Problem:** Low water pressure or flow
- **Solution:** Check pressure regulator, clean filter
- **Prevention:** Regular filter maintenance

**Problem:** Uneven water coverage
- **Solution:** Adjust nozzle positions, check for clogs
- **Prevention:** Use filtered water, regular nozzle cleaning

**Problem:** Leaking connections
- **Solution:** Tighten connections, replace worn fittings
- **Prevention:** Use quality fittings, proper installation

#### Environmental Issues
**Problem:** Excessive heat from LEDs
- **Solution:** Improve ventilation, consider lower wattage LEDs
- **Prevention:** Monitor temperature during design

**Problem:** High humidity or condensation
- **Solution:** Improve ventilation, adjust irrigation timing
- **Prevention:** Proper environmental control design
