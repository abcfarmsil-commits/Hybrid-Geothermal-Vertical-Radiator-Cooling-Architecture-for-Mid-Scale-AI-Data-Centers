# **Module 3 — Germination Engines**

*(Student-Facing Visual Handout)*

---

## **Diagram 1 — What a “Germination Engine” Is**

┌─────────────────────────────┐  
│     GERMINATION ENGINE      │  
│  (Tray-Based Biological Unit│  
│   with Controlled Inputs)   │  
└─────────────────────────────┘  
            ▲  
            │  
   ┌────────┼────────┐  
   │        │        │  
 Seed     Water    Light

**Key idea:**  
This is **not a crop**.  
It is a **biological engine** that converts inputs into predictable early-stage plant growth.

---

## **Diagram 2 — Tray as a System (Not a Container)**

     ┌──────────────────────┐  
      │        LIGHT         │  
      │  (fixed cycle / band)│  
      └──────────────────────┘  
                │  
                ▼  
┌────────────────────────────────┐  
│           TRAY BODY             │  
│                                 │  
│  ┌───────────┐                  │  
│  │   SEED    │  ← biological    │  
│  └───────────┘     driver       │  
│                                 │  
│  ┌───────────┐                  │  
│  │  MEDIA    │  ← water buffer  │  
│  └───────────┘                  │  
│                                 │  
│  (optional drainage holes)      │  
└────────────────────────────────┘  
                │  
                ▼  
         Root Mat / Biomass

**Teaching point:**  
Drainage, color, media type \= **tuning knobs**, not requirements.

---

## **Diagram 3 — Inputs → Controls → Outputs**

INPUTS              CONTROLS              OUTPUTS  
\---------------------------------------------------------  
Seed        ──▶  Light Cycle   ──▶  Edible Biomass  
Water       ──▶  Temp Band     ──▶  Root Mat  
Time        ──▶  Humidity      ──▶  Spent Media

**Important:**  
There is **no yield promise** here — only *repeatability*.

---

## **Diagram 4 — Clean vs Dirty Logic (Germination Stage)**

CLEAN SIDE  
───────────────  
• Seed handling  
• Tray prep  
• Early growth  
• Observation

        │  
        │  (time \+ biology)  
        ▼

DIRTY SIDE  
───────────────  
• Root mass  
• Spent media  
• Transfer to:  
  \- food harvest  
  \- fungi feed  
  \- vermiculture

**Why this matters:**  
Contamination risk is **managed by separation**, not chemicals.

---

## **Diagram 5 — Rail / Cart Flow Concept**

       │  
       ▼

┌──────────┐   ┌──────────┐   ┌──────────┐

       (same conditions, different start times)

**Teaching point:**  
Movement is logistical, **not biological**.

## **Diagram 6 — Why Trays May Differ (Case-by-Case)**

┌─────────────┐     ┌─────────────┐  
│  Solid Tray │     │ Drain Tray  │  
│             │     │ (drilled)   │  
│ • basil     │     │ • peas      │  
│ • amaranth │     │ • sunflowers│  
└─────────────┘     └─────────────┘

**Rule:**  
Uniform hardware is nice.  
**Correct biology is better.**

---

## **Diagram 7 — Connection to the Larger System**

GERMINATION ENGINES  
        │  
        ├──▶ Direct Harvest (food)  
        │  
        ├──▶ Fungi Substrate Input  
        │  
        └──▶ Vermiculture Feedstock

**This is the pivot:**  
Microgreens stop being the *product*  
and become **the front end of a regenerative loop**.

---

## **One-Sentence Takeaway for Students**

A germination engine is a controlled biological starter unit whose value is predictability, not yield.
