## Naming Conventions

### Files
- Use kebab-case for all filenames: `document-name.md`
- Use descriptive names: `hydro-battery-concept.md` not `doc1.md`
- Include version numbers if needed: `model-fit-map-v2.md`

### Folders
- Use numbered prefixes for ordering: `01-Executive-Overview/`
- Use kebab-case for folder names
- Keep folder names concise but descriptive

## Git Workflow

### Initial Commit
```bash
git init
git add README.md
git add REPOSITORY_STRUCTURE_GUIDE.md
git commit -m "Initial commit: Repository structure and README"
```

### Organized Commit
```bash
# After moving all documents
git add .
git commit -m "Organize repository: Move 152+ documents into structured folders"
```

### Push to GitHub
```bash
git branch -M main
git remote add origin https://github.com/abcfarmsil-commits/ABCFARMSIL.git
git push -u origin main
```
