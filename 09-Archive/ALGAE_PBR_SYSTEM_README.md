# Fresh Algae PBR Platform - 53-ft Reefer Container

**A gravity-first, 5-lane photobioreactor system for fresh multi-flavor algae production.**

---

## 🌿 System Overview

This system transforms a 53-foot refrigerated container into a high-efficiency, fresh-algae production platform optimized for culinary and nutraceutical applications. Unlike biofuel-focused systems, this platform prioritizes:

- **Fresh product quality** with 7-10 day shelf life
- **Multi-flavor capability** through strain and stress management
- **Cold-zone continuity** from harvest through delivery
- **Modular lane isolation** for operational flexibility

### Key Metrics

- **Total Working Volume:** ~1,200 L
- **Daily Production:** ~1-2 kg fresh paste/day total
- **Per-Flavor Output:** ~200-400 g per day
- **Power Requirement:** ~12-18 kW peak
- **Cooling:** Base load handled by reefer unit
- **Footprint:** 53-ft container (~48-49 ft usable length)

---

## 🎨 5-Lane Flavor Strategy

Each lane operates independently, enabling:

1. **Neutral Green (Base)** - Standard algae profile
2. **Umami/Savory** - Enhanced savory compounds
3. **Bright/Golden** - Lighter, citrusy notes
4. **Deep Green/Protein** - High-protein variant
5. **Seasonal/R&D/Chef Collab** - Experimental varieties

### Operational Flexibility

- **Pause individual lanes** without stopping the system
- **Stress one lane** while others remain stable
- **Swap strains** in one lane at a time
- **Custom formulations** for restaurant partnerships

## 🏗️ System Architecture

### Zone Breakdown

| Zone | Percentage | Length | Function |
|------|------------|--------|----------|
| **Zone A - PBR Growth (Clean)** | ~53% | ~26 ft | All live culture |
| **Zone B - Harvest/Wash** | ~27% | ~13 ft | Concentration + rinse |
| **Zone C - Utilities/Ops** | ~20% | ~10 ft | CO₂, pumps, controls |

### PBR Configuration

- **Type:** Vertical flat-panel PBRs, double-sided LED
- **Material:** Food-grade polymer or glass
- **Layout:** 5 lanes × 2 panels per lane
- **Panel Volume:** ~120 L per panel
- **Total Panels:** 10
- **Mounting:** Parallel to container length with central service aisle

### Lane Isolation

Each lane has:
- Dedicated media feed
- Dedicated CO₂ injection
- Dedicated harvest outlet
- Shared but valved CIP return

**No flavor bleed. No operational gymnastics.**

---

## 🔄 Process Flow

### Harvest Train

1. **Low-shear membrane concentration**
2. **Cold rinse** to strip media taste
3. **Immediate chill** to 0-4°C
4. **Packaging** in cold-chain format

### Sanitation

- **CIP (Clean-in-Place)** system with valved returns
- **Food-grade sanitation** built-in
- **USP-grade equipment** where applicable
- **Regular testing** for quality verification

## ⚙️ Technical Specifications

### Utilities

- **Power:** ~12-18 kW peak (lights primary variable)
- **Cooling:** Reefer unit handles base load
- **CO₂:** Food-grade bulk or manifolded cylinders
- **Water:** Low net usage (mostly rinse + CIP)
- **Drainage:** Clean → boundary → solids (inspector-friendly)

### Lighting

- **Type:** Double-sided LED arrays
- **Spectrum:** Tunable for strain optimization
- **Intensity:** Adjustable per lane
- **Control:** Independent photoperiod zones

### Controls

- **PLC/Edge Controller:** Local deterministic control
- **Telemetry Export:** MQTT/OPC-UA with local buffer
- **Sensors:** Temperature, pH, DO, flow rate, light intensity
- **HMI:** Local tablet/panel interface for operators
